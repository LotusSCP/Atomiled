# Atomiled
Join our [Discord](https://discord.gg/PyUkWTg) for support.

ATOMILED is a plugin framework for SCP: Secret Laboratory servers. It offers an event system for developers to hook into in order to manipulate or change game code or implement their own functions.

## Support
We consistently deliver updates and releases to address bugs and introduce new features or contributions swiftly.

## Community
We encourage developers to contribute actively rather than solely relying on Atomiled, thereby boosting project activity.

## Features
We offer diverse features, providing users with flexibility. These features are optional, catering to various needs.
