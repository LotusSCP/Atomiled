# Manual Installation

You can download Atomiled manually following this steps:

### Pick a release

You can select a release inside [our official GitHub repo](https://github.com/Exmod-Team/ATOMILED/releases/).

### Download the release

Download the `Atomiled.tar.gz` file and extract it with your favourite tool, we recommend [7Zip](https://www.7-zip.org/) or [WinRar](https://www.win-rar.com/download.html?&L=6).

### Installation

1. Move the file **``Assembly-CSharp.dll``** to: **`(Your Server Folder)/SCPSL_Data/Managed`** and replace the file.
2. Move the **``ATOMILED``** folder to **`%appdata%`**

#### Note:
- Windows: This **`%appdata%`** folder is the one located inside **`AppData/Roaming`**, not just `AppData`.
- Linux: This **`%appdata%`** folder is the one called **`~/.config`**.
